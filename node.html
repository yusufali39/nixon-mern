Node.js is a powerful platform for developing server-side JavaScript. Node.js is a platform built on Chrome's JavaScript runtime for easily building fast and scalable network applications.


Note: Node.js is not a Language


Then What is Node.js?

  • Node.js uses Chrome's V8 engine to excute code

  • Code and syntax is very similar to JavaScript but not exactly the same.

  • Node.js is an open source server environment

  • Node.js is free

  • Node.js runs on various platforms (Windows, Linux, Unix, Mac OS X, etc.)

  • Node.js can connect with databse and add, delete, modify data in your database


    • What Can Node.js Do?

  • Node.js can generate dynamic page content

  • Node.js can create, open, read, write, delete, and close files on the server

  • Node.js can collect form data

  • Node.js can add, delete, modify data in your database


    • Why Do We Use Node.js?

  • Node.js mostly used for API

  • With API we can connect the same databse with Web App, Mobile App

  • Node.js is easy to understand who knows JavaScript

  • With Node.js and JavaScript, you can become full stack developer

  • Node.js is free and open source
